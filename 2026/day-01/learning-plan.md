# 90 Days DevOps Learning Plan

I am a fresher and MCA student with basic knowledge of programming and web development. I want to build strong DevOps and Cloud skills to become job-ready.

---

## My Understanding of DevOps
DevOps is a combination of development and operations that helps teams build, test, deploy, and manage applications faster and more efficiently using automation and collaboration.

Cloud engineering helps in deploying and managing applications on cloud platforms like AWS.

---

## Why I Am Learning DevOps
I want to become a skilled DevOps Engineer and improve my technical skills in Linux, Docker, Kubernetes, CI/CD, and Cloud technologies.

---

## My 3 Goals for the Next 90 Days
1. Learn Linux, Git, Docker, Kubernetes, and AWS fundamentals.
2. Build and deploy projects using Docker and Kubernetes.
3. Become confident in DevOps tools and workflows for internships and jobs.

---
